Open Source License Introduction and Analysis


    • Introduction

      Introduce most of the commonly used, mainstream licenses, including providing detailed text of the terms and conditions, and generalizing the terms and conditions for easy reading. In this page, you can read the outlines of more than thirty major license, and get the highly generalized information about conditions, limitations and permission of each license in a table. If you want to read the full text of one license, just click the hyperlink under the table. Also, you can search licenses in the search box located at the top of page, fuzzy search function is supported.

    • Compliance Analysis

      Analyse the content of uploaded file or input text with our license extraction algorithm. If the content matches a license, the algorithm would return the license name, otherwise it would return none. You can also upload a folder, then all files in the folder would be analyzed, and we show the folder structure with a retractable tree, the file name and its license name would be shown as the leaves of this tree. The compliance information between all license included in the folder would be shown in the compliance box. In this page, click any license name, the page would jump to the introduction of this license.

    • Conflict Analysis

      Most function in this conflict page are same as compliance page. But in conflict page, you can only upload one folder, then the license extraction algorithm would analyse all file in this folder and extract its license name if matched, otherwise none. The structure of the folder would be shown as a tree with the leaves of file name and license name. Click the license name, you can get the text content of the file, and the license-related content is highlighted. The conflict relationship between all licenses included in this folder are shown in the conflict analysis box.
